== MySQL Compatibility

PuzzleDB supports the MySQL protocol via https://github.com/cybergarage/go-mysql[go-mysql], a framework for implementing MySQLâ€‘compatible servers in Go.

image::https://raw.githubusercontent.com/cybergarage/go-mysql/main/doc/img/framework.png[]

The framework handles protocol and system commands; PuzzleDB focuses on implementing core DDL and DML query handling.
