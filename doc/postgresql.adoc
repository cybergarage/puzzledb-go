== PostgreSQL Compatibility

PuzzleDB supports the PostgreSQL protocol via https://github.com/cybergarage/go-postgresql[go-postgresql], a framework for implementing PostgreSQLâ€‘compatible servers in Go.

image::https://raw.githubusercontent.com/cybergarage/go-postgresql/master/doc/img/framework.png[]

The framework handles protocol and system commands; PuzzleDB focuses on implementing primary DDL and DML query processing.
