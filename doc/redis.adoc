== Redis Compatibility

PuzzleDB supports the Redis protocol via https://github.com/cybergarage/go-redis[go-redis], a framework for implementing Redisâ€‘compatible servers in Go.

image::https://raw.githubusercontent.com/cybergarage/go-redis/main/doc/img/framework.png[]

The framework handles protocol and system commands (e.g., PING, SELECT); PuzzleDB implements user commands (e.g., SET, GET).
